# Automation Suite (Python File Backup System)

## Overview
This project is a Python-based automation tool that performs safe file backups from a source directory to a destination folder. It demonstrates real-world automation concepts such as modular design, file handling, logging, CLI interaction, and testing.

---

## Features
- CLI-based interface
- Automated file backup system
- Modular architecture (separated components)
- Logging system for tracking operations
- Error handling for safe execution
- Pytest support for basic validation
- Timestamped backup folders

---

---

## How It Works
1. User runs `main.py`
2. CLI menu appears
3. Backup system reads files from `source/`
4. Files are copied to timestamped backup folder
5. Logs are recorded in `logs/log.txt`
